# 專案摘要

結合 Mocha + Chai + Sinon ，架構 NodeJS 測試平台。


# 事前準備

使用本專案的產出物，您電腦的作業環境，須先備妥下列軟體，完成應有之安裝：

 * git
 * node
 * npm
 
 
# 安裝作業

循下列操作程序，完成本專案作業環境之安裝工作。
 
```
 $ git clone git@github.com:AlanJui/mocha-test.git

 $ cd mocha-test

 $ npm install
```


# 執行作業

執行以下指令。

```
 $ npm test
```
